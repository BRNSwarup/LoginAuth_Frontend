# 🎨 Frontend - Authentication System

## 📌 Overview

This is the frontend of a full-stack authentication system built using React. It provides a responsive user interface for user registration, login, and password reset using OTP verification.

---

## 🚀 Features

* 🔑 User Signup & Login UI
* 🔐 JWT-based authentication handling
* 📧 OTP verification for password reset
* 🔄 Form validation 
* ⚡ Responsive design using Bootstrap

---

## 🛠️ Tech Stack

* React.js
* Bootstrap
* Axios (for API requests)
